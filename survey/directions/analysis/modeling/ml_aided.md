# ML/LLM 辅助密码分析

## 方向概述

利用机器学习（尤其是深度学习和大语言模型）辅助密码分析。2019 年 Gohr 的开创性工作首次展示神经网络可以学习超越经典差分的密码区分器，引发了该领域的研究热潮。目前正在向 LLM 驱动的自动化分析演进。

## 关键问题

- 神经区分器的可解释性与泛化能力
- ML 辅助搜索 vs 传统求解器的优势边界
- LLM 在密码分析策略生成中的能力
- 训练数据的获取与标注
- 从特定密码到通用框架的泛化

## 当前前沿 (2024-2026)

- LLM 作为密码分析策略引擎（CryptoLLM）
- 强化学习驱动的搜索策略
- GNN 用于密码结构分析
- 可解释 AI 在密码分析中的应用

## 与 CryptoLLM 的关联

- **核心相关**: CryptoLLM 本身就是这个方向的实践
- **已验证**: LLM 研究能力实验 — 两个 LLM 通过空集证明测试
- **竞争分析**: 需要密切跟踪最新进展
